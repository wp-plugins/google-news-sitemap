== Installation ==

Download, Upgrading, Installation:

Upgrade

1. First deactivate Google News Sitemap
1. Remove the `google-news-sitemap` directory

**Install**

1. Unzip the `google-news-sitemap.zip` file. 
1. Upload the the `google-news-sitemap` folder (not just the files in it!) to your `wp-contents/plugins` folder. If you're using FTP, use 'binary' mode.

**Activate**

1. In your WordPress administration, go to the Plugins page
1. Activate the Google News Sitemap plugin and a subpage for Google News Sitemap will appear in your Options menu.

If you find any bugs or have any ideas, please mail me.
